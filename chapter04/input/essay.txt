Visual Studio Code is an editor first and foremost and includes the features you need for highly productive source code editing.
This topic takes you through the basics of the editor and helps you get moving with your code.
Being able to keep your hands on the keyboard when writing code is crucial for high productivity.
VS Code has a rich set of default keyboard shortcuts as well as allowing you to customize them.
Keyboard Shortcuts Reference - Learn the most commonly used and popular keyboard shortcuts by downloading the reference sheet.
Install a Keymap extension - Use the keyboard shortcuts of your old editor (such as Sublime Text, Atom, and Vim) in VS Code by installing a Keymap extension.
          
Customize Keyboard Shortcuts - Change the default keyboard shortcuts to fit your style.
